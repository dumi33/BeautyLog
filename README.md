# � BeautyLog (뷰티로그)

**"매일 더 아름다워지는 당신의 기록, 뷰티로그"**

BeautyLog는 피부과 시술 내역, 피부 상태 변화, 그리고 회복 과정을 체계적으로 기록하고 관리할 수 있는 여성을 위한 맞춤형 스마트 뷰티 다이어리입니다.

---

## 📱 주요 기능 (Key Features)

### 1. 홈 대시보드 (Home Dashboard)
- **최근 시술 요약**: 가장 최근에 받은 시술과 시술 후 경과 일수(D-day)를 한눈에 확인.
- **맞춤형 추천 가이드**: 시술 후 회복 단계(진정, 보습, 재생 등)에 맞는 관리 팁 실시간 제공.
- **회복 상태 트래킹**: 시각화된 카드로 현재 회복 단계를 직관적으로 표시.

### 2. 세밀한 기록 작성 (Detailed Logging)
- **날짜 및 시술 정보**: 캘린더 피커를 통한 날짜 선택, 방문 병원 및 시술명 상세 입력.
- **회복 상태 관리**: '회복 중', '거의 회복', '완료' 3단계 상태 지정.
- **증상 레벨 체크**: 붓기, 멍, 통증, 열감 등 4가지 주요 상태를 5단계 강도로 기록.
- **시술 전후 사진 및 메모**: 최대 5장의 사진 업로드와 자유로운 시술 후기 메모 기능.

### 3. 스마트 캘린더 (Smart Calendar)
- **주간/월간 뷰**: 기록된 데이터를 도트(Dot)와 아이콘으로 표시하여 전체 관리 흐름을 파악.
- **자기 만족 스탬프 (♥)**: 외모가 마음에 드는 날을 체크하여 긍정적인 피부 변화 추적.
- **상세 데이터 조회**: 날짜별 기록된 내용을 캘린더에서 바로 확인 가능.

### 4. 사용자 계정 및 동기화 (Auth & Sync)
- **소셜 로그인**: Google 계정을 통한 간편한 로그인 및 자동 회원가입.
- **클라우드 데이터 저장**: 작성한 모든 기록은 안전하게 클라우드에 저장되어 기기 변경 시에도 유지.

## 🛠 기술 스택 (Tech Stack)

### Frontend
- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router 기반)
- **Library**: [React](https://reactjs.org/)
- **Styling**: Vanilla CSS (iOS 스타일의 미니멀 프리미엄 디자인)

### Backend & Database
- **BaaS**: [Supabase](https://supabase.com/) (PostgreSQL)
- **Authentication**: [NextAuth.js](https://next-auth.js.org/) (Google Auth)

## 🎨 디자인 컨셉
- **Clean & Minimal**: 불필요한 장식을 배제하고 정보 전달에 집중한 화이트/소프트 그레이 베이스 UI.
- **Point Color**: 여성스러운 프리미엄 감성을 담은 로즈 골드(`--point: #e08b8b`) 컬러 포인트.
- **Micro-interactions**: 버튼 호버, 모달 팝업 등 부드러운 애니메이션으로 고급스러운 사용자 경험 제공.
