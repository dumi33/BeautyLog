# 💅 BeautyLog

여성을 위한 뷰티 관리 앱

## 📱 소개

BeautyLog는 여성들의 머리와 피부 관리를 체계적으로 기록하고 추적할 수 있도록 돕는 애플리케이션입니다.

## ✨ 주요 기능

### 🧴 피부 관리
- 일일 스킨케어 루틴 기록
- 피부 상태 트래킹
- 사용 중인 제품 관리
- 피부 변화 사진 기록

### 💇‍♀️ 헤어 관리
- 헤어 스타일 변경 히스토리
- 염색/펌 주기 관리
- 사용 제품 기록
- 미용실 방문 일정 관리

## 🎯 개발 목표

편리하고 직관적인 UI/UX를 통해 사용자가 쉽게 자신의 뷰티 루틴을 관리하고 개선할 수 있도록 지원합니다.
